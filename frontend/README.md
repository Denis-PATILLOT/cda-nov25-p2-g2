# Gettting Started

## Install dependencies

```
npm i
```

## Setup environment variables

```
cp .env.example .env
```

And then adjust variables in ```.env``` to match your local environment.

## Start in dev mode

```
npm run dev
```

## Build

```
npm run build
```

## Start in production mode

```
npm run build
```

## Launch Codegen for creating/updating schema.ts

```
npm run codegen
```

## Launch unit tests for frontend

```
npm run test
```
